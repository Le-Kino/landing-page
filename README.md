# LeKino — lekino.ca

Landing page for [LeKino](https://lekino.ca), a Quebec nonprofit promoting healthy habits and physical activity through community events.

Built as a static bilingual (FR/EN) single-page site, hosted on GitHub Pages at [lekino.ca](https://lekino.ca).

## Stack

Pure HTML + CSS — no build step, no dependencies (except Google Fonts via CDN).

## Contributing

Edit `index.html` directly. Push to `main` → GitHub Pages deploys automatically.
